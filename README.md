# Number-Game

Number-Game is game number like guitar hero. You can run it in browser.

## Installation

Use the package manager npm for install dependencies :

```bash
git clone https://github.com/azizrosyid/Number-Game
cd Number-Game
npm install 
```

## Usage
For running in localhost use :
```bash
npm run start-dev
```

Build app :
```bash
npm run build
```
